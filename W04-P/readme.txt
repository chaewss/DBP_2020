정보관리 측면에서 하나의 테이블에 모든 정보를 넣는 것을 좋은 방법이 아니기 때문에 관계형 데이터베이스를 사용한다.
관계형 데이터베이스의 핵심은 테이블 간의 관계를 맺어주는 것인데 이는 JOIN을 이용해 연결해준다.

HTML코드 중 변하는 부분은 PHP코드로 작성하고 변하지 않는 부분은 HTML코드로 작성한다.

HTML코드의 table 태그 중 <tr>:행 / <td>:열 / <th>:제목줄(가운데 정렬, 진하게)

PHP코드에서 사용하는 header는 어떤 동작을 직접적으로 수행해준다.
헤더를 이용해 '이전 페이지로 돌아가기'같이 따로 클릭할 필요 없이 바로바로 반영되니까 훨씬 편리하고 좋았다.

저번 수업에도 배웠지만 버튼을 직접 클릭하는 것이 링크보다 자동적으로 처리하기가 불편해 보안이 강화된다.
